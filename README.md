# SMSE-Whmcs
這是用於 WHMCS 8.8 的速買配(SMSE)金流串接套件<br>
本插件僅於 WHMCS 8.8 版本測試無誤，其他版本不確定<br>
下載之後請放入/modules/gateways<br>
<br>
## 設定說明<br>
務必先匯入 smse_pay_record.sql 進資料庫，<br>
若無法匯入，則依據檔案內容創建對應 table 以及 column，<br>
若沒有匯入，則會造成插件異常。<br>
